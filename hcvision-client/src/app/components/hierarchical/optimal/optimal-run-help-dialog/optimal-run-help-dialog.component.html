Instructions

Follow these steps to utilize the page effectively:

1

Select your Dataset:

Use the dropdown menu to choose the dataset you want to work with.

2

Choose Processing Option:

Decide whether you want to process the entire dataset or just a sample to find the Hierarchical optimal parameters.

3

Select Attributes:

Indicate which attributes from the dataset you want the algorithm to consider. Use the provided options.

4

Click Run:

Once you've configured your preferences, click the "Run" button to initiate the process of finding the optimal parameters.
