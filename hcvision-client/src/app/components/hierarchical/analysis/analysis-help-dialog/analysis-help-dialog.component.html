Running Hierarchical Clustering Analysis

Follow these steps to run hierarchical clustering analysis:

1

Select Dataset:

Choose the dataset you want to analyze from the available options.

2

Select Linkage:

Choose the linkage method that best suits your analysis (e.g., single, complete, average).

3

Select Number of Clusters:

Specify the desired number of clusters you want the algorithm to create. This determines how many distinct groups the algorithm should identify in the data.

4

Select Dataset Sampling:

Indicate whether you want to use the entire dataset or a sample for the analysis.

5

Select Attributes:

Choose the attributes from the dataset that you want the algorithm to consider.

6

Click Run:

Once you've configured your analysis preferences, click the "Run" button to initiate hierarchical clustering.

7

Results:

Review the clustering results based on the selected parameters. The following outcomes are available:

  • Dendrogram: Visual representation of the hierarchical clustering structure.
  • Parallel Coordinates Graph: Visualization of high-dimensional data showing relationships between attributes.
  • Cluster Assignments: Display of the dataset with assigned clusters for each data point.

Notes:

  • Linkage: Linkage is the method used to calculate the distance between clusters. Common linkage methods include single, complete, and average linkage.
  • Number of Clusters: Selecting the number of clusters determines how many groups the hierarchical clustering algorithm will identify in the data. It's a crucial parameter that influences the granularity of the clustering result.
  • Dendrogram: A dendrogram is a tree diagram that represents the hierarchical clustering of the data. It visually displays the arrangement of clusters and their relationships.
  • Parallel Coordinates: Parallel coordinates is a method for visualizing high-dimensional data by representing each data point as a line and connecting its coordinates with parallel lines.
